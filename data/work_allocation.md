# 작업 할당

> 각 영상의 담당자와 작업 상태를 관리하는 파일입니다.

## 작업 현황

| 영상 폴더 | 담당자 | 상태 | 시작일 | 완료일 | 비고 |
|----------|--------|------|--------|--------|------|
| video1   | -      | 대기 | -      | -      | -    |
| video2   | -      | 대기 | -      | -      | -    |
| video3   | -      | 대기 | -      | -      | -    |

## 상태 코드

- **대기**: 아직 작업 시작 안 함
- **진행중**: 현재 작업 중
- **완료**: 어노테이션 완료
- **검수중**: 작업 완료 후 검수 진행 중
- **확정**: 최종 검수 완료

## 사용 방법

1. 새로운 영상 작업을 시작할 때 담당자와 시작일을 기록
2. 작업이 완료되면 상태를 업데이트하고 완료일을 기록
3. Git commit & push하여 팀원과 공유

## 예시

```markdown
| 영상 폴더 | 담당자 | 상태 | 시작일 | 완료일 | 비고 |
|----------|--------|------|--------|--------|------|
| video1   | 김철수 | 진행중 | 2026-04-17 | -      | 프레임 500장 |
| video2   | 이영희 | 완료 | 2026-04-17 | 2026-04-18 | 프레임 300장 |
| video3   | 박민수 | 대기 | -      | -      | -    |
```
